¡Bienvenidos!

A continuación se muestran las listas de distribución de Mailman que se anuncian públicamente en lists.lug.fi.uba.ar. Déle con el ratón a un nombre de lista para visitar las páginas de configuración de esa lista. Para configurar una lista que no se enseña públicamente, abra una URL similar a la que existe, pero con un '/' seguido por el nombre de la lista. Si tiene la autorización adecuada, también puede crear una lista de distribución nueva.

Se puede encontrar información general sobre las listas en la página de información general de las listas de distribución.
