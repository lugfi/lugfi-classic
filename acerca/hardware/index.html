:: Hardware del LUGFI ::

phi.lug.fi.uba.ar es el servidor principal del LUGFI, el que provee todos los servicios (está página web, mail, gforge, jabber, wiki, el cliente web de IRC, etc.).

Este servidor existe gracias al aporte de mucha gente que, de una u otra forma, donó tiempo, dinero o el mismo hardware para hacerlo posible. Este es el hardware de phi a la fecha (con su respectivo donante):

  • Mother Asus M2N-E9
  • Athlon X2 3800EE
  • Gabinete Satellite, con 2 coolers y una buena fuente
  • 1GB(2x512Mb) Memoria RAM DDR2 800 CAS 4-4-4-12
  • Disco de 80GB, donado por Maxy y Marga
  • Disco de 40GB, préstamo del LABI
