:: Documentos del LUGFI ::

Acá es donde vamos publicando todos los documentos, tutoriales, FAQs, etc, realizados por el LUGFI.

:: Documentos disponibles ::
  • FAQ General del LUGFI: En este documento se recopilan las preguntas más frecuentes realizadas en la lista de correo con sus respectivas respuestas.
    Ver online (wiki)
  • FAQ sobre SubVersion: Este documento pretende dar una breve introducción al uso de Subversion.
    Ver online | HTML | PDF | LaTeX | LyX
  • Configuración de un Gateway: Explica detalladamente los conceptos relevantes y los pasos necesarios para configurar una máquina con GNU/Linux como gateway de Internet.
    Ver online | HTML | PDF | LaTeX
  • TP genérico en LaTeX: Un documento LaTeX con ejemplos útiles que puede servir como base para hacer un trabajo práctico o un informe.
    PDF | Código fuente
  • Tablas en LaTeX: Tutorial con todo lo que necesitás saber acerca de cómo crear tablas en LaTeX.
    PDF | LaTeX | Bibliografía LaTeX
  • Glosarios en LaTeX: Tutorial con todo lo que necesitás saber acerca de cómo crear glosarios en LaTeX.
    PDF | LaTeX | ZIP
  • Herramientas para ingeniería: Una lista de las herramientas libres para ingeniería disponibles.
    Ver online | HTML | PDF | LaTeX
  • Introducción a los sistemas de control de versiones: Este documento explíca los conceptos básicos teóricos a la hora de manejar código fuente de forma ordenada, presentando como caso de estudio de dos modelos diferentes (distribuido y centralizado) a darcs y subversion, dando ejemplos de ambos y explicando cómo montar un repositorio público de darcs en aleph.
    Ver online | HTML
  • Charla sobre Filosofia e Ingenieria: Charla sobre la relacion y los puntos de interes en comun entre la filosofia y la ingenieria.
    PDF | ODP | SXI
