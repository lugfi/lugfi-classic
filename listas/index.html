:: Listas de correo ::

Estas son las listas de correo del LUGFI:

  • lug (en) listas.fi.uba.ar : Es la lista principal del LUGFI, acá podés hacer todas las preguntas que tengas sobre GNU/Linux y/o aportar tus conocimientos, ya sea contestando otras preguntas o simplemente comentando algo interesante. Es importante no tener miedo de preguntar, la lista está para eso.
  • programacion (en) listas.fi.uba.ar : Se trata de una lista dedicada a todo lo que tiene que ver con la programación, especialmente orientada al desarrollo de software libre. Todas las preguntas relacionadas con desarrollo de software son bienvenidas.
  • bar (en) lug.fi.uba.ar : Esta lista es para organizar salidas, mandar chistes, anécdotas, historias, fotos, etc. Es para tener un lado más "social" del Lugfi, sin llenar de Off-Topic la lista principal.
  • devel (en) lug.fi.uba.ar : Esta lista es para temas relacionados con desarrollo de proyectos de software abierto, sin importar la plataforma en la que se trabaje. O sea, no importa para qué sistema operativo o con qué lenguaje desarrollen, mientras se trate de desarrollos abiertos/libres (FLOSS).
  • org (en) lug.fi.uba.ar : En esta lista se discute y coordina todo lo que tenga que ver con la organización interna del LUG. Es decir, los temas relacionados con el sitio web, la organización de eventos, reuniones, cursos, etc. Si tenés ganas de participar, estás invitado; y lo primero que podés hacer es anotarte en esta lista.
  • noticias (en) lug.fi.uba.ar : Para recibir por correo todas las noticias relacionadas con el LugFI (novedades, eventos, etc), inscribite en esta lista. Es una lista de bajo volumen de tráfico.
  • trabajo (en) lug.fi.uba.ar : En esta lista se publican ofertas/busquedas laborales relacionadas a GNU/Linux y Software Libre. Podés suscribirte para recibir las ofertas o para enviar si conocés alguna.
  • seguridad (en) lug.fi.uba.ar : Esta es una lista expresamente dedicada a discutir sobre seguridad en el software surgida a partir de una charla brindada por la gente de Corest Community en la facultad.
