:: Servidor Jabber del LUGFI ::

El LUGFI cuenta con un servidor Jabber de acceso público. Para usarlo, sólo tenés que sacar tu cuenta usuario@lug.fi.uba.ar utilizando tu cliente Jabber favorito.

:: ¿Pero qué es Jabber? ::

Jabber es un protocolo abierto para el intercambio en tiempo real de mensajes, que ofrece una funcionalidad similar a la de otros sistemas como AIM, ICQ, MSN Messenger y Yahoo. Las principales características de Jabber son:

  • Es abierto -- la especificación de Jabber es abierta y libre, y lo seguirá siendo por siempre. Es gracias a ello que existen múltiples implementaciones de servidores y clientes Jabber.
  • Es estándar -- el sistema Jabber utiliza un protocolo estándar, XMPP, desarrollado por la Jabber Software Foundation.
  • Es extensible -- usando la potencia del lenguaje XML, es fácil extender el protocolo Jabber para agregarle una funcionalidad determinada.
  • Es descentralizado -- el sistema completo no depende del buen funcionamiento de un servidor central, sino que está distribuido en muchos servidores; y cualquiera puede montar su propio servidor Jabber.
  • Es seguro -- cualquier servidor Jabber puede ser aislado de la red pública Jabber, se utiliza SSL para las comunicaciones cliente-servidor y numerosos clientes soportan pgp-gpg para encriptar las comunicaciones cliente a cliente.

Entre las características más interesantes de Jabber se destaca la posibilidad de utilizar transportes. Los transportes permiten a un usuario Jabber conectarse a otros sistemas de mensajería instantánea utilizando el mismo cliente. Esto significa que podés comunicarte con tus amigos que aun utilizan MSN o cualquier otro protocolo, utilizando una cuenta Jabber.

Jabber puede crear confusión en un principio respecto a otros sistemas de mensajería instantánea porque habitualmente, en otros IM, se identifica el cliente con el protocolo. En el caso de Jabber esto no es así: existe un protocolo y cada uno de los Clientes es una implementación.

¡Empezá a usar Jabber hoy mismo! Elegí un cliente de la lista de clientes disponibles, elegí un servidor (entre los que obviamente se encuentra el nuestro), create una cuenta, ¡y listo!
